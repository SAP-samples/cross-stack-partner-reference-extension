# Changes

## December 2025 (Initial Version)

The tutorials and sample cross-stack partner reference application code are available in the main branch.

### Highlights

- End-to-end cross-stack extension scenario integrating SAP S/4HANA Cloud Public Edition with a side-by-side SAP BTP CAP multi-tenant application.
- Step-by-step tutorials for building, deploying, and integrating both on-stack and side-by-side components.
- Ready-to-consume sample code for the Cross-Stack Partner Reference Extension.
- Guidance on secure integration, multi-off delivery, and best practices for ABAP RAP.

### Tutorials Provided

1. Developing an on-stack extension in SAP S/4HANA Cloud Public Edition with ABAP RAP.
2. Configuring a side-by-side multi-tenant application using SAP BTP CAP.
3. Setting up secure integration between SAP S/4HANA Cloud Public Edition and SAP BTP.
4. Implementing value help, custom fields, and actions in the on-stack extension.
5. Managing multi-off delivery using gCTS and Git integration.
6. Reviewing the end-to-end solution in the development system.
