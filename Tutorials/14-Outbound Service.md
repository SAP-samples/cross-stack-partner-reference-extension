# Creating an Outbound Service 

To create an outbound service, follow these steps:

1. Right-click the `ZPRA_PSE_SALES_ORDER` package and choose **New > Other ABAP Repository Object**.

2. Search for `Outbound Service`, select it, and choose **Next**.

3. Enter the following data and choose **Next**:

    - **Name**: `ZPRA_PSE_SO_PS_OB_REST`

    - **Description**: `Poetry Slam Update`

    - **Service Type**: `HTTP Service`


4. Select a transport request and choose **Finish**.
